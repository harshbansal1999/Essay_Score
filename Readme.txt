This is a Natural Language Processing Project 
It includes train data and test dataset which includes essay with their respective ids.
Each essay have been given score by rater1 and rater2 in train dataset
Now, we have to predict th score of rater1 and rater2 for test data set
Project include:
1.text.R file which is source code as coding is done in Rstudio
2.test_set and training_set_rel3 are datasets
3.rater1_pred_final is the prediction file by rater1 and rater2_pred_final is the prediction file by rater2
